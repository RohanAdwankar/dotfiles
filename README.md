# macos dotfiles

`bash macos.sh` will install the programs I use and ideally set the relevant preferences and configs.

This repo is incomplete because several configuration files are not present: 

Tmux config is set by `tmuxtui --config`

Nvim config is set by `https://github.com/RohanAdwankar/nvim` 

Cargo preference is set by `https://github.com/RohanAdwankar/rohanrust`

These should eventually be incorporated in such that running the script sets up the computer exactly.
